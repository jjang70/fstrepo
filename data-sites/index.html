공개 데이터셋

  • Kaggle Datasets - 다양한 머신러닝 데이터셋 제공

  • 공공데이터포털 - 대한민국 공공 데이터
  • 국가통계포털 - 국가 통계 데이터 제공
  • 한국산업안전보건공단_산업안전보건 실태조사(제조업) 원시자료 - 산업안전보건 실태조사 데이터
  • 교육통계서비스 - 교육 통계 데이터 제공
  • UCI Machine Learning Repository - 머신러닝 연구용 데이터셋
  • Google Dataset Search - 데이터셋 검색 엔진

API 제공 사이트

  • JSONPlaceholder - 테스트용 REST API
