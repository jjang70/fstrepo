# Python 데이터 분석 가이드

GitHub Pages를 사용하여 호스팅되는 Python 데이터 분석 환경 설정 및 오픈소스 라이브러리 사용 가이드 웹사이트입니다.

## 프로젝트 소개

이 프로젝트는 Python 데이터 분석을 시작하는 분들을 위한 종합 가이드를 제공합니다. Jupyter 환경 설정부터 주요 라이브러리 사용법, 머신러닝 개념, 자료형 설명까지 체계적으로 정리되어 있습니다.

## 프로젝트 구조

```
.

├── jupyter/                # Jupyter 설정
│   └── index.html
├── library/                # 오픈소스 라이브러리
```

## 주요 기능

### 1. Jupyter 환경 설정
- Python 다운로드 링크
- Jupyter 환경 자동 설정 스크립트 제공

### 2. 오픈소스 라이브러리 가이드
- **pandas**: 데이터 분석 라이브러리 (merge, concat, join 등 상세 설명)
- **matplotlib**: 데이터 시각화
- **requests**: HTTP 요청 처리
- **beautifulsoup4**: 웹 스크래핑

### 3. 머신러닝 개념
- 지도학습 (Supervised Learning)
- 비지도학습 (Unsupervised Learning)
- 강화학습 (Reinforcement Learning)

### 4. 자료형 설명
- 기본 자료형 (int, float, str, bool)
- 시퀀스 자료형 (list, tuple, str)
- 매핑 자료형 (dict)
- 집합 자료형 (set, frozenset)
- 배열 자료형 (NumPy Array) - 주요 메서드 및 함수 상세 설명
- 시리즈 자료형 (Pandas Series) - 주요 메서드 상세 설명

### 5. 데이터 수집 사이트
- 공개 데이터셋 사이트 목록
- API 제공 사이트
- 웹 스크래핑 연습 사이트
- 특수 목적 데이터 사이트

## 사용 방법

1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 브라우저에서 열어 확인할 수 있습니다.
3. GitHub Pages를 활성화하면 자동으로 웹사이트가 호스팅됩니다.

## GitHub Pages 설정

1. GitHub 저장소의 Settings로 이동
2. Pages 섹션으로 이동
3. Source에서 브랜치 선택 (보통 `main` 또는 `master`)
4. 저장 후 몇 분 후에 `https://[사용자명].github.io/[저장소명]` 주소로 접속 가능
