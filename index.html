pandas 사용방법

1. 데이터 읽기/쓰기

import pandas as pd

# CSV 파일 읽기
df = pd.read_csv('data.csv')

# Excel 파일 읽기
df = pd.read_excel('data.xlsx')

# CSV 파일 저장
df.to_csv('output.csv', index=False)

# Excel 파일 저장
df.to_excel('output.xlsx', index=False)

2. 데이터 선택/필터링

# 열 선택
df['column_name']
df[['col1', 'col2']]

# 행 선택
df.loc[0]  # 인덱스로 선택
df.iloc[0]  # 위치로 선택

# 조건 필터링
df[df['column'] > 100]
df[(df['col1'] > 10) & (df['col2'] < 50)]

3. 데이터 변환

# 열 추가
df['new_column'] = df['col1'] + df['col2']

# 열 삭제
df = df.drop('column_name', axis=1)

# 데이터 타입 변경
df['column'] = df['column'].astype(int)

# 결측치 처리
df = df.dropna()  # 결측치 제거
df = df.fillna(0)  # 결측치를 0으로 채우기

4. 그룹화 및 집계

# 그룹화
grouped = df.groupby('column_name')

# 집계 함수
df.groupby('column').sum()
df.groupby('column').mean()
df.groupby('column').count()

# 여러 집계 함수 동시 적용
df.groupby('column').agg({
    'col1': 'sum',
    'col2': 'mean',
    'col3': 'count'
})

requests 사용방법

1. 기본 사용법

import requests

# GET 요청
response = requests.get('https://api.example.com/data')
print(response.status_code)
print(response.json())

# POST 요청
data = {'key': 'value'}
response = requests.post('https://api.example.com/data', json=data)

2. 헤더 및 파라미터

# 헤더 추가
headers = {'User-Agent': 'Mozilla/5.0'}
response = requests.get('https://example.com', headers=headers)

# 쿼리 파라미터
params = {'page': 1, 'limit': 10}
response = requests.get('https://api.example.com', params=params)

# 쿠키 사용
cookies = {'session': 'abc123'}
response = requests.get('https://example.com', cookies=cookies)

3. 파일 다운로드

# 파일 다운로드
url = 'https://example.com/file.pdf'
response = requests.get(url)

with open('file.pdf', 'wb') as f:
    f.write(response.content)

# 스트리밍 다운로드 (대용량 파일)
response = requests.get(url, stream=True)
with open('large_file.zip', 'wb') as f:
    for chunk in response.iter_content(chunk_size=8192):
        f.write(chunk)

4. 에러 처리

try:
    response = requests.get('https://example.com', timeout=5)
    response.raise_for_status()  # HTTP 에러 확인
except requests.exceptions.RequestException as e:
    print(f"에러 발생: {e}")

# 상태 코드 확인
if response.status_code == 200:
    print("성공")
elif response.status_code == 404:
    print("페이지를 찾을 수 없습니다")

머신러닝

좌측 메뉴에서 머신러닝의 주요 분류를 선택하여 개념을 확인하세요.

지도학습 (Supervised Learning)

개념

지도학습은 입력 데이터와 정답 레이블이 함께 제공되는 학습 방법입니다. 모델은 입력과 출력 간의 관계를 학습하여 새로운 데이터에 대한 예측을 수행합니다.

주요 카테고리

  • 분류 (Classification): 이산적인 클래스 레이블을 예측하는 문제
  • 회귀 (Regression): 연속적인 수치 값을 예측하는 문제

대표 알고리즘

  • 선형 회귀 (Linear Regression)
  • 로지스틱 회귀 (Logistic Regression)
  • 의사결정 트리 (Decision Tree)
  • 랜덤 포레스트 (Random Forest)
  • 서포트 벡터 머신 (SVM)
  • 신경망 (Neural Networks)

비지도학습 (Unsupervised Learning)

개념

비지도학습은 정답 레이블 없이 입력 데이터만으로 패턴을 찾아내는 학습 방법입니다. 데이터의 숨겨진 구조나 분포를 발견하는 데 사용됩니다.

주요 카테고리

  • 클러스터링 (Clustering): 유사한 데이터를 그룹으로 묶는 작업
  • 차원 축소 (Dimensionality Reduction): 고차원 데이터를 저차원으로 변환
  • 연관 규칙 학습 (Association Rule Learning): 데이터 간의 연관성을 찾는 작업

대표 알고리즘

  • K-평균 클러스터링 (K-Means)
  • 계층적 클러스터링 (Hierarchical Clustering)
  • 주성분 분석 (PCA)
  • t-SNE
  • 자기 조직화 지도 (SOM)

강화학습 (Reinforcement Learning)

개념

강화학습은 에이전트가 환경과 상호작용하며 보상 신호를 통해 학습하는 방법입니다. 시행착오를 통해 최적의 행동 정책을 학습합니다.

주요 구성 요소

  • 에이전트 (Agent): 학습하고 행동하는 주체
  • 환경 (Environment): 에이전트가 상호작용하는 외부 세계
  • 상태 (State): 환경의 현재 상황
  • 행동 (Action): 에이전트가 취할 수 있는 동작
  • 보상 (Reward): 행동에 대한 피드백

대표 알고리즘

  • Q-Learning
  • Deep Q-Network (DQN)
  • 정책 경사 (Policy Gradient)
  • Actor-Critic
  • Proximal Policy Optimization (PPO)

자료형

좌측 메뉴에서 Python의 주요 자료형 카테고리를 선택하여 세부분류와 요약사항을 확인하세요.

기본 자료형

세부분류

  • 숫자형 (Numeric Types)
    • int - 정수형
    • float - 실수형
    • complex - 복소수형
  • 문자열 (String) - str
  • 불린 (Boolean) - bool

요약사항

기본 자료형은 Python에서 가장 기본이 되는 데이터 타입입니다. 숫자형은 수학적 연산에 사용되며, 정수(int), 실수(float), 복소수(complex)로 구분됩니다. 문자열(str)은 텍스트 데이터를 저장하고 처리하는 데 사용되며, 불변(immutable) 객체입니다. 불린(bool)은 True와 False 두 가지 값만 가지며, 조건문과 논리 연산에 필수적입니다.

시퀀스 자료형

세부분류

  • 리스트 (List) - list
    • 가변(mutable) 시퀀스
    • 순서가 있는 요소들의 모음
    • 다양한 자료형 혼합 가능
  • 튜플 (Tuple) - tuple
    • 불변(immutable) 시퀀스
    • 리스트보다 빠른 접근 속도
    • 딕셔너리 키로 사용 가능
  • 문자열 (String) - str
    • 불변(immutable) 시퀀스
    • 문자들의 나열

요약사항

시퀀스 자료형은 순서가 있는 요소들의 집합입니다. 리스트는 가변적이어서 요소 추가, 삭제, 수정이 가능하며, 가장 많이 사용되는 자료구조입니다. 튜플은 불변적이어서 한 번 생성되면 수정할 수 없지만, 리스트보다 메모리 효율적이고 빠른 접근이 가능합니다. 문자열도 시퀀스의 일종으로, 인덱싱과 슬라이싱이 가능합니다.

매핑 자료형

세부분류

  • 딕셔너리 (Dictionary) - dict
    • 키-값 쌍(key-value pair) 구조
    • 가변(mutable) 자료형
    • 키는 불변 자료형만 사용 가능
    • 순서 보장 (Python 3.7+)

요약사항

딕셔너리는 키(key)와 값(value)의 쌍으로 데이터를 저장하는 자료구조입니다. 해시 테이블을 기반으로 하여 평균 O(1) 시간 복잡도로 조회가 가능합니다. 키는 불변 자료형(문자열, 숫자, 튜플 등)만 사용할 수 있으며, 값은 모든 자료형을 사용할 수 있습니다. Python 3.7부터는 삽입 순서가 보장되어 순서가 있는 매핑 자료형이 되었습니다.

집합 자료형

세부분류

  • 집합 (Set) - set
    • 가변(mutable) 집합
    • 중복 요소 불가
    • 순서 없음
    • 집합 연산 지원
  • 불변 집합 (Frozen Set) - frozenset
    • 불변(immutable) 집합
    • 딕셔너리 키로 사용 가능
    • 다른 집합의 요소로 사용 가능

요약사항

집합 자료형은 수학의 집합 개념을 구현한 자료구조로, 중복된 요소를 허용하지 않습니다. 순서가 없으며 해시 테이블을 기반으로 하여 빠른 멤버십 테스트가 가능합니다. 합집합, 교집합, 차집합 등의 집합 연산을 지원합니다. set은 가변적이어서 요소 추가/삭제가 가능하지만, frozenset은 불변적이어서 딕셔너리의 키나 다른 집합의 요소로 사용할 수 있습니다.

배열 자료형

세부분류

  • NumPy 배열 (ndarray) - numpy.ndarray
    • 다차원 배열 구조
    • 동일한 데이터 타입만 저장
    • 벡터화 연산 지원
    • 메모리 효율적
  • 배열 특징
    • shape, dtype 속성
    • 브로드캐스팅 지원
    • 인덱싱 및 슬라이싱
    • 수학 연산 최적화

요약사항

NumPy 배열(ndarray)은 과학 계산과 데이터 분석의 핵심 자료구조입니다. 리스트와 달리 동일한 데이터 타입만 저장하며, C로 구현되어 매우 빠른 연산 속도를 제공합니다. 다차원 배열을 지원하여 행렬 연산, 이미지 처리, 머신러닝 등에 널리 사용됩니다. 벡터화 연산을 통해 반복문 없이도 전체 배열에 대한 연산을 수행할 수 있어 성능이 우수합니다.

시리즈 자료형

세부분류

  • Pandas 시리즈 (Series) - pandas.Series
    • 1차원 레이블이 있는 배열
    • 인덱스와 값의 쌍
    • 다양한 데이터 타입 지원
    • NaN 값 처리 가능
  • 시리즈 특징
    • 인덱스 기반 접근
    • 벡터화 연산 지원
    • 결측치 처리 메서드
    • 데이터프레임의 열(column)로 사용

요약사항

Pandas Series는 1차원 레이블이 있는 배열로, 딕셔너리와 리스트의 특징을 결합한 자료구조입니다. 각 값에 인덱스가 연결되어 있어 이름으로 데이터에 접근할 수 있습니다. NumPy 배열을 기반으로 하되, 레이블 인덱싱과 결측치 처리 기능이 추가되었습니다. 데이터프레임의 기본 구성 요소이며, 시계열 데이터, 실험 결과 등 순서가 있는 데이터를 다루는 데 적합합니다.
