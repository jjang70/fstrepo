요약사항

NumPy 배열(ndarray)은 과학 계산과 데이터 분석의 핵심 자료구조입니다. 리스트와 달리 동일한 데이터 타입만 저장하며, C로 구현되어 매우 빠른 연산 속도를 제공합니다. 다차원 배열을 지원하여 행렬 연산, 이미지 처리, 머신러닝 등에 널리 사용됩니다. 벡터화 연산을 통해 반복문 없이도 전체 배열에 대한 연산을 수행할 수 있어 성능이 우수합니다.

주요 메서드 및 함수

1. shape / reshape()

기능: 배열의 차원 확인 / 배열 형태 변경
입력: shape (튜플 또는 정수)
출력: 튜플 (shape) / ndarray (reshape)
예시: arr.shape, arr.reshape(2, 3)
실무 활용: 이미지 데이터 전처리, 행렬 변환, 차원 조정

2. dtype / astype()

기능: 데이터 타입 확인 / 데이터 타입 변환
입력: dtype (문자열 또는 타입)
출력: dtype 객체 / ndarray (변환된 타입)
예시: arr.dtype, arr.astype('float64')
실무 활용: 메모리 최적화, 연산 타입 맞추기, 데이터 형식 변환

3. np.mean() / np.sum() / np.std()

기능: 통계 함수 (평균, 합계, 표준편차)
입력: array, axis=None, keepdims=False
출력: 스칼라 또는 배열
예시: np.mean(arr), np.sum(arr, axis=0)
실무 활용: 데이터 요약 통계, 특성 엔지니어링, 정규화

4. np.max() / np.min() / np.argmax() / np.argmin()

기능: 최대/최소값 및 인덱스 찾기
입력: array, axis=None
출력: 스칼라 또는 배열 (값/인덱스)
예시: np.max(arr), np.argmax(arr, axis=1)
실무 활용: 이상치 탐지, 최적값 찾기, 분류 예측

5. np.where()

기능: 조건에 따라 값 선택
입력: condition, x (True일 때), y (False일 때)
출력: ndarray (조건에 맞는 값)
예시: np.where(arr > 0, arr, 0)
실무 활용: 조건부 데이터 처리, 마스킹, 데이터 필터링

6. np.concatenate() / np.stack()

기능: 배열 결합 / 배열 쌓기
입력: arrays (튜플), axis=0
출력: ndarray (결합된 배열)
예시: np.concatenate([arr1, arr2]), np.stack([arr1, arr2])
실무 활용: 데이터 병합, 배치 데이터 구성, 특성 결합

7. np.zeros() / np.ones() / np.empty()

기능: 특정 값으로 초기화된 배열 생성
입력: shape, dtype=float
출력: ndarray (초기화된 배열)
예시: np.zeros((3, 4)), np.ones((2, 2), dtype=int)
실무 활용: 배열 초기화, 마스크 생성, 결과 저장소 준비

8. np.arange() / np.linspace()

기능: 연속된 숫자 배열 생성
입력: start, stop, step (arange) / start, stop, num (linspace)
출력: ndarray (숫자 배열)
예시: np.arange(0, 10, 2), np.linspace(0, 1, 100)
실무 활용: 인덱스 생성, 그리드 생성, 샘플링 구간 설정

9. np.random

기능: 랜덤 배열 생성 (random, randint, normal 등)
입력: size, dtype 등
출력: ndarray (랜덤 값)
예시: np.random.rand(3, 4), np.random.normal(0, 1, 100)
실무 활용: 데이터 증강, 가중치 초기화, 시뮬레이션

10. np.dot() / np.matmul()

기능: 행렬 곱셈 연산
입력: a, b (배열)
출력: ndarray (행렬 곱 결과)
예시: np.dot(A, B), np.matmul(A, B)
실무 활용: 선형 대수 연산, 신경망 계산, 변환 행렬

11. np.transpose() / .T

기능: 배열 전치 (행과 열 바꾸기)
입력: axes (전치 순서)
출력: ndarray (전치된 배열)
예시: arr.T, np.transpose(arr)
실무 활용: 행렬 연산, 데이터 형태 변환, 특성 행렬 변환

12. np.unique()

기능: 배열의 고유값 반환
입력: array, return_index=False, return_counts=False
출력: ndarray (고유값) 또는 튜플
예시: np.unique(arr), np.unique(arr, return_counts=True)
실무 활용: 중복 제거, 범주 확인, 레이블 인코딩

13. 브로드캐스팅

기능: 서로 다른 크기의 배열 간 연산 자동 확장
입력: 서로 다른 shape의 배열
출력: ndarray (연산 결과)
예시: arr + 10, arr1 + arr2 (shape 호환 시)
실무 활용: 벡터화 연산, 정규화, 효율적인 계산

14. 인덱싱 및 슬라이싱

기능: 배열의 특정 요소 또는 부분 추출
입력: 인덱스 또는 슬라이스
출력: ndarray (추출된 부분)
예시: arr[0, 1], arr[1:3, :], arr[arr > 5]
실무 활용: 데이터 추출, 필터링, 부분 데이터 분석
