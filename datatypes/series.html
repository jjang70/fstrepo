요약사항

Pandas Series는 1차원 레이블이 있는 배열로, 딕셔너리와 리스트의 특징을 결합한 자료구조입니다. 각 값에 인덱스가 연결되어 있어 이름으로 데이터에 접근할 수 있습니다. NumPy 배열을 기반으로 하되, 레이블 인덱싱과 결측치 처리 기능이 추가되었습니다. 데이터프레임의 기본 구성 요소이며, 시계열 데이터, 실험 결과 등 순서가 있는 데이터를 다루는 데 적합합니다.

주요 메서드

1. head() / tail()

기능: 처음/마지막 n개 데이터 확인
입력: n (기본값 5)
출력: Series (처음/마지막 n개)
예시: series.head(10), series.tail()
실무 활용: 대용량 데이터의 구조 파악, 데이터 샘플링 확인

2. describe()

기능: 통계 요약 정보 제공 (개수, 평균, 표준편차, 최소/최대, 사분위수)
입력: 없음 또는 percentiles
출력: Series (통계 요약)
예시: series.describe()
실무 활용: 데이터 탐색 분석(EDA), 이상치 감지, 분포 파악

3. value_counts()

기능: 값의 빈도수 계산 (내림차순 정렬)
입력: normalize=False, sort=True, ascending=False
출력: Series (값: 빈도수)
예시: series.value_counts(), series.value_counts(normalize=True)
실무 활용: 범주형 데이터 분석, 빈도 분석, 데이터 분포 확인

4. isnull() / notnull()

기능: 결측치(NaN) 여부 확인
입력: 없음
출력: Series (Boolean)
예시: series.isnull(), series.notnull()
실무 활용: 결측치 탐지, 데이터 품질 검사, 필터링 조건으로 사용

5. fillna()

기능: 결측치를 지정한 값으로 채우기
입력: value (스칼라, dict, Series, method='ffill'/'bfill')
출력: Series (결측치 채워진 결과)
예시: series.fillna(0), series.fillna(method='ffill')
실무 활용: 결측치 처리, 시계열 데이터 전처리, 평균/중앙값으로 대체

6. dropna()

기능: 결측치가 있는 행 제거
입력: axis=0, how='any'/'all'
출력: Series (결측치 제거된 결과)
예시: series.dropna()
실무 활용: 결측치가 많은 데이터 정리, 분석 전 데이터 클렌징

7. sort_values()

기능: 값 기준 정렬
입력: ascending=True/False, na_position='last'/'first'
출력: Series (정렬된 결과)
예시: series.sort_values(), series.sort_values(ascending=False)
실무 활용: 데이터 정렬, 상위/하위 데이터 추출, 순위 분석

8. unique() / nunique()

기능: 고유값 반환 / 고유값 개수
입력: 없음
출력: ndarray (고유값) / int (개수)
예시: series.unique(), series.nunique()
실무 활용: 범주 확인, 중복 제거, 데이터 카디널리티 파악

9. apply()

기능: 각 요소에 함수 적용
입력: func (함수), convert_dtype=True
출력: Series (변환된 결과)
예시: series.apply(lambda x: x*2), series.apply(str.upper)
실무 활용: 데이터 변환, 형식 변환, 복잡한 계산 로직 적용

10. map()

기능: 값 매핑 (딕셔너리 또는 함수 사용)
입력: arg (dict, Series, 함수)
출력: Series (매핑된 결과)
예시: series.map({'A': 1, 'B': 2}), series.map(str.lower)
실무 활용: 레이블 인코딩, 카테고리 변환, 값 치환

11. groupby()

기능: 그룹별 집계 연산
입력: by (그룹핑 기준), axis=0
출력: GroupBy 객체
예시: df.groupby('category')['price'].sum()
실무 활용: 카테고리별 통계, 그룹별 분석, 집계 연산

12. str 접근자

기능: 문자열 메서드에 접근 (문자열 Series 전용)
입력: 문자열 메서드 (upper, lower, split, contains 등)
출력: Series (문자열 처리 결과)
예시: series.str.upper(), series.str.contains('pattern')
실무 활용: 텍스트 전처리, 패턴 매칭, 문자열 변환
