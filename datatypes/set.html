집합 자료형

세부분류

  • 집합 (Set) - set
    • 가변(mutable) 집합
    • 중복 요소 불가
    • 순서 없음
    • 집합 연산 지원
  • 불변 집합 (Frozen Set) - frozenset
    • 불변(immutable) 집합
    • 딕셔너리 키로 사용 가능
    • 다른 집합의 요소로 사용 가능

요약사항

집합 자료형은 수학의 집합 개념을 구현한 자료구조로, 중복된 요소를 허용하지 않습니다. 순서가 없으며 해시 테이블을 기반으로 하여 빠른 멤버십 테스트가 가능합니다. 합집합, 교집합, 차집합 등의 집합 연산을 지원합니다. set은 가변적이어서 요소 추가/삭제가 가능하지만, frozenset은 불변적이어서 딕셔너리의 키나 다른 집합의 요소로 사용할 수 있습니다.
